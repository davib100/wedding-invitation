# **App Name**: Eternal Vows Invitation

## Core Features:

- Interactive Envelope Opening: Simulates opening a realistic envelope with a wax seal, transitioning to the digital invitation.
- Animated Landing Page Transition: Smooth, cinematic zoom effect from the envelope to the invitation content.
- RSVP Form: Modal form for guests to confirm their attendance with name, contact and date/time of confirmation.
- Background Music Control: Allows admin to upload background music.
- Admin Panel Authentication: Hidden admin panel accessible via Google OAuth for managing invitation content and guest list.
- Guest List Management: Admin tool to view the list of guests, sort by their answers to any field in the form, and change invitation details such as the bride and groom names or the music, as well as to update data and texts for the app.
- Event Countdown: Displays a countdown timer showing the time remaining until the wedding.

## Style Guidelines:

- Primary color: Soft gold (#D4AF37) to convey luxury and romance.
- Background color: Off-white with paper texture (#F5F5DC) for a classic, refined feel.
- Accent color: Deep red (#8B0000) for the wax seal and interactive elements, evoking passion.
- Headline font: 'Playfair', a modern serif with an elegant feel.
- Body font: 'PT Sans', a humanist sans-serif offering warmth and readability. 
- Use minimalist, elegant icons that complement the luxurious aesthetic. 
- Clean, airy layout with ample spacing to enhance sophistication.
- Subtle, slow animations for interactions (envelope opening, page transitions) to maintain a premium, cinematic feel.
